Textarea adalah elemen pada HTML yang biasa digunakan untuk menampilkan baris program. Pada kondisi default, ukuran textarea dapat diatur ulang oleh pengguna. Untuk beberapa kebutuhan, textarea diharuskan untuk ukurannya tidak dapat diubah oleh pengguna.

Berikut ini penampakan dari textarea yang bisa diatur ukurannya.

Pada gambar diatas dapat lihat bahwa terdapat pengaturan ukuran pada textarea secara default. Untuk menghilangkannya anda bisa menggunakan program berikut.

Berikut adalah penampakan textarea setelah setelah property resize diatur menjadi none

Lihat juga penggunaan animation-timing-function ini pada pembuatan animasi fade-in dan juga animasi pindah tempat
