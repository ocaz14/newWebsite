Tipe data Integer adalah jenis tipe data yang dapat digunakan pada operasi matematika, sementara tipe data String tidak dapat digunakan pada operasi matematika.

Jika anda menjumlahkan 1 + 1 dimana keduanya ber-tipe data integer, maka hasilnya adalah 2. Tapi juga anda menjumlahkan 1 + 1 dimana keduanya ber-tipe data String, maka hasilnya adalah 11. Dan anda mungkin akan memperoleh hasil ERROR jika anda memaksakan menggunakan operator matematika.

Belum lagi jika operator matematika digunakan pada dua nilai dengan tipe data berbeda maka anda akan memperoleh hasil ERROR.

Untuk itu, dibutuhkan baris program dimana mengubah nilai dari tipe data String menjadi tipe data Integer.

Untuk mengubah tipe data string menjadi integer, dapat menggunakan fungsi parseInt(). Berikut adalah contoh penulisan program dari fungsi ini.
