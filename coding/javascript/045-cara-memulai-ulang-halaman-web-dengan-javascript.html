Javascript juga dapat digunakan untuk me-refresh / memulai ulang halaman website. Pada kesempatan kali ini, akan dibuat tombol dimana pada saat tombol diklik halaman akan melakukan refresh.

Membuat tombol dengan HTML

Tombol dapat dibuat dengan tag "button". Anda juga dapat membuat tombol dengan jenis tag yang lain seperti tag "div". Jenis tag apapun dapat digunakan, selama tag tersebut dapat menjalankan fungsi pada saat diklik.

Gunakan property "onclick" untuk menuliskan nama fungsi yang akan dijalankan. Fungsi ditulis dengan nama fungsi diikuti dua tanda kurung. Misalnya pada contoh penulisan diatas menggunakan fungsi dengan nama reload().

Fungsi

Setelah tombol sudah dibuat, selanjutnya adalah membuat / deklarasi fungsi. Fungsi ini berisi program yang akan dijalankan yaitu memulai ulang halaman.

Program akan memulai ulang halaman pada saat fungsi dijalankan. Sementara itu, fungsi dapat dijalankan dengan menekan tombol yang telah dibuat sebelumnya. Sehingga, untuk memulai ulang halaman web, dapat dengan menekan tombol yang telah dibuat.

Hasil

Berikut tampilan tombol yang telah dibuat beserta fungsinya.
