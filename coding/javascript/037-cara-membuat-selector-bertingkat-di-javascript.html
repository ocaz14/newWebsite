Selector adalah metode yang digunakan untuk memilih atau menentukan elemen yang akan dimodifikasi pada sebuah halaman website (document).

Selector juga dapat digunakan bertingkat dimana elemen dipilih dari hasil selector sebelumnya. Berikut adalah contoh penulisan programnya.

Variabel Array "class_a" berisi sekumpulan elemen dengan nama class "a" pada sebuah halaman penuh (document).

Variabel Array "class_b" berisi sekumpulan elemen dengan nama class "b" yang merupakan anggota dari variabel "class_a" dimana pada halaman HTML berada di dalam elemen dengan nama class "a".

Elemen dengan nama class "b" diluar elemen dengan nama class "a" tidak manjadi anggota dari variabel array "class_b".
