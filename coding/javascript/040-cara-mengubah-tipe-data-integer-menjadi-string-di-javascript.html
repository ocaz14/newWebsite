Tipe data integer berbeda dengan tipe data string. Tipe data Integer adalah jenis tipe data yang dapat digunakan pada operasi matematika, sementara tipe data String tidak dapat digunakan pada operasi matematika.

Di beberapa kesempatan, terkadang dibutuhkan data dengan tipe data string. Untuk mengubah tipe data integer menjadi tipe data string digunakan fungsi toString(). Berikut ini contoh cara penulisan program dengan menggunakan fungsi toString().

Pada contoh diatas, variabel int_nilai_1 merupakan variabel dengan tipe data integer. Fungsi toString() mengambil nilai dari variabel int_nilai_1 kemudian mengubahnya menjadi nilai dengan tipe data string yang kemudian disimpan pada variabel nilai_1
