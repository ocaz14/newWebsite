Untuk menambah "class" pada sebuah elemen, hal yang harus dilakukan pertama adalah membuat javascript mengenali elemen tersebut.

Untuk mengenali elemen tersebut, dapat menggunakan selector "class" atau selector "id" atau jenis selector yang lain.

Selector Class

Berikut adalah contoh dimana digunakan selector "class" untuk javascript dapat mengenali elemen tersebut.

Jika anda ingin menambah nama class yang sama pada beberapa elemen, maka sebaiknya anda menggunakan selector class atau selector lain yang sifatnya menyimpan elemen terpilih di dalam sebuah array.
