Javascript dapat digunakan untuk mengatur ulang elemen yang terdapat pada HTML. Salah satu yang dapat diatur adalah judul halaman web.

Document.title

Selector adalah sebuah perintah di javascript yang digunakan untuk menyambungkan HTML dan javascript. Dengan ini, anda bisa mengatur kandungan, dan bahkan penampilan pada elemen yang terpilih.

Selector yang bisa digunakan untuk dapat mengatur judul halaman adalah adalah selector "title". Berikut adalah contoh penulisannya.

Dengan baris program diatas, javascript akan mencari elemen dengan tag "title" pada document (halaman web) dan mengatur ulang kandungnnya menjadi "Judul Baru"
